# AGENTS.md 规则遵守要求

## 规则说明

在开始任何开发任务之前，必须检查当前项目根目录是否存在 `AGENTS.md` 文件。

### 强制要求

1. **必须阅读并遵守 AGENTS.md**：如果项目根目录存在 `AGENTS.md` 文件，必须完整阅读其内容，并在整个开发过程中严格遵守其中的所有规则和要求。

2. **任务完成时检查完善性**：在任务完成时，需要审视 `AGENTS.md` 的内容是否完善。如果发现：
   - 有新添加的功能模块未在 `AGENTS.md` 中记录
   - 有新的目录结构或文件组织方式未说明
   - 有新的开发流程或操作指南未涵盖
   - 有新的依赖关系或配置要求未描述

   则应当主动更新 `AGENTS.md`，补充缺失的内容，确保文档始终与项目实际状态保持一致。

3. **文档更新原则**：
   - 保持与现有文档风格一致
   - 使用简体中文编写
   - 遵循现有的章节结构和格式
   - 更新相关的架构图和依赖关系图（如有）

### 执行流程

```
开始任务
    ↓
检查 AGENTS.md 是否存在
    ↓
存在 → 阅读 AGENTS.md → 遵守规则执行任务
    ↓
不存在 → 正常执行任务
    ↓
任务完成
    ↓
检查 AGENTS.md 是否需要更新
    ↓
需要 → 更新 AGENTS.md → 任务完成
不需要 → 任务完成
```
